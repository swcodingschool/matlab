## 언어 기본 사항  
* MATLAB은 "Matrix Laboratory"의 약어이다. 
다른 프로그래밍 언어에서는 주로 한 번에 하나의 숫자를 다루지만, MATLAB®은 행렬이나 배열 단위로 연산을 수행한다. 
언어 기본 사항에서는 변수 생성, 배열 인덱싱, 산술 연산, 데이터형과 같은 기본 작업을 다룬다.

## 프로그래밍  
* 반복해서 수행하거나 나중에 참조할 수 있도록 저장하려는 명령 시퀀스가 있으면 프로그램 파일에 저장해 둘 수 있다. 
가장 간단한 유형의 MATLAB® 프로그램은 스크립트이다. 스크립트에는 명령줄에 입력하는 것과 같은 일련의 명령을 저장해 둘 수 있다.
프로그래밍의 유연성을 더하고자 한다면 입력값을 받아 출력값을 반환하는 함수를 만든다.
특화된 데이터 구조가 있거나 특수한 종류의 데이터와 상호 작용하기 위해 많은 함수가 필요한 경우에는 
객체 지향 프로그래밍 기법을 사용하여 클래스를 만드십시오.  
* 프로그래밍 [바로가기](https://kr.mathworks.com/help/matlab/programming-and-data-types.html?s_tid=CRUX_lftnav)  
>  스크립트 / 함수 / 라이브 스크립트 및 라이브 함수 / 클래스 / 파일과 폴더 / 프로그래밍 유틸리티  

## 앱 작성  
* 앱은 코드에 대한 간단한 포인트-앤-클릭 인터페이스를 제공하는 자체 충족형(Self-contained) MATLAB® 프로그램이다.
앱은 상호 작용하면서 명령을 실행시킬 수 있는 대화형 컨트롤을 포함하며, 그 종류는 메뉴, 버튼, 슬라이더 같은 것들이다. 
또한 앱은 데이터 시각화나 대화형 데이터 탐색을 위한 플롯을 포함하고 있다. 
앱은 패키징하여 다른 MATLAB 사용자와 공유할 수도 있고, MATLAB Compiler™를 사용하여 독립 실행형 응용 프로그램으로 배포할 수도 있다.
* 앱 작성 [바로가기](https://kr.mathworks.com/help/matlab/gui-development.html?s_tid=CRUX_lftnav)  
> 앱 디자이너 / GUIDE 또는 프로그래밍 워크플로 / 앱을 패키징하고 공유하기   

## 소프트웨어 개발 툴  
* 프로젝트의 크기와 복잡도가 증가하면서 그에 맞게 MATLAB®은 협업 소프트웨어 개발 업무를 지원하는 기능을 제공한다.
예를 들어, MATLAB 파일을 Git™ 또는 Subversion® 소스 컨트롤 시스템과 연동하거나 직접 작성한 코드의 기능과 성능을 테스트할 수 있다.
코드를 다른 사람과 공유하려면 프로젝트나 다른 파일을 툴박스로 패키징한다. 
* 소프트웨어 개발 툴 [바로가기](https://kr.mathworks.com/help/matlab/software-development.html?s_tid=CRUX_lftnav)  
> 디버그 및 분석 / 성능과 메모리 / 프로젝트 / 소스 콘트롤 연동 / 테스트 프레임워크 / 툴박스 배포  

## 외부 언어 인터페이스  
* MATLAB®은 다른 프로그래밍 언어와의 유연한 양방향 통합을 제공하므로, 사용자는 레거시 코드를 재사용할 수 있다.
사용하는 응용 프로그램에 적합한 MATLAB 기능을 선택하는 데 도움을 받으려면 MATLAB과 외부 프로그래밍 언어 및 시스템 통합하기 항목을 참조한다.
